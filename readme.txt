Connect Namaste! LMS with GamiPress

== Description ==

Gamification for [Namaste! LMS](https://wordpress.org/plugins/namaste-lms/ "Namaste! LMS") through [GamiPress](https://wordpress.org/plugins/gamipress/ "GamiPress"): earn points for achieving milestones in Namaste! LMS.

This plugin adds new activity events to GamiPress

= New Events =

= For Courses =

* When a student finish a course
* When a student finish a specific course

= For Modules (When Modules are Enabled) =

* When a student user finish a module
* When a student finish a specific module
* When a student finish a module of a specific course

= For Lessons =

* When a student complete a lesson
* When a student complete a specific lesson
* When a student complete a lesson of a specific course

== Installation ==

1. Unzip the contents and upload the entire `namaste-gamipress` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Namaste! LMS Bridge to GamiPress" in your menu and manage the plugin
